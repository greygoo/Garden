Control

Lights:
cl00 - turn off light 0
cl01 - turn on light 0

cl10 - turn off light 1
cl11 - turn on light 1


Pumps:
cp00 - turn off pump 0
cp01 - turn on pump 0

cp10 - turn off pump 1
cp11 - turn on pump 1


Fans:
cf00 - turn off fan 0
cf01 - turn on fan 0

cf10 - turn off fan 1
cf11 - turn on fan 1

cf20 - turn off fan 2
cf21 - turn on fan 2

cs** - set speed for Fan 0 from ** = 00 - 99


Sensors

Temperature

st0 - print temperature sensor 0 temperature
st1 - print temperature sensor 1 temperature

sh  - print humidity sensor humidity
sd  - print humidity sensor temperature 

se  - print TDS sensor TDS value
sc  - calibrate TDS sensor

sv  - get total volume of flowmeter
sr  - get flow rate from last pump action 

spX - get state of pump X = int
